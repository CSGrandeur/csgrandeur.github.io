发表于 2017-03-07 更新于 2021-03-31 Disqus：

花了一个多星期给CSUACM做了个新网站。

acm.csu.edu.cn

这几天太累了也不想打太多字了，做篇日志留念。也许什么时候有空了再写个新的Judge端吧。
